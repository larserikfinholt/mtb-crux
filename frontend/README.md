# MTB-Crux Frontend

This is the frontend for the MTB-Crux application, which allows users to register and compete with mountain bike crux challenges.

## Features

- Interactive map showing crux locations
- Detailed crux information with difficulty ratings
- Mobile-responsive design
- List view of all crux points

## Components

### CruxMap

The CruxMap component displays an interactive Leaflet map with all crux locations. Features include:

- Map markers for each crux point
- Popup information on click
- Mobile-friendly layout
- Interactive list view that centers the map on selected crux

## Development

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Setup

1. Install dependencies:
```
npm install
```

2. Run development server:
```
npm run dev
```

3. Build for production:
```
npm run build
```

## Technologies

- Vue 3 with Composition API
- TypeScript
- Leaflet for interactive maps
- Pinia for state management
